---

## Version Classification

Vector currently defines **six primary system versions**, grouped by accessibility focus:

### Core Versions
1. Vector Basic
2. Vector Pro
3. Vector Vision (for visually impaired users)

### Extended Accessibility Versions
4. Vector Hearing (for deaf and hard-of-hearing users)
5. Vector Sense (for deafblind users)
6. Vector Motion (for users with paralysis or severe motor limitations)

---

## 1. Vector Basic

**Target users:** General users without disabilities  
**Purpose:** Cognitive assistance, safety, focus, situational awareness

### Key Features
- On-device computer vision
- Context-aware visual cues
- Attention prioritization
- Environmental alerts
- Minimal UI overlays

### Use Cases
- Urban navigation
- Work and study assistance
- Digital fatigue reduction
- Everyday safety support

---

## 2. Vector Pro

**Target users:** Professionals, developers, researchers  
**Purpose:** Advanced interaction, extended control, system customization

### Additional Features
- Expanded AI context modeling
- Advanced user configuration
- Developer and research tools
- Multi-modal input/output
- External accessory support

### Use Cases
- Engineering
- Medical environments
- Research and field work
- High-risk professions

---

## 3. Vector Vision

**Target users:** Blind and visually impaired users  
**Purpose:** Environmental perception through non-visual channels

### Key Features
- Audio-based spatial guidance
- Object and obstacle detection
- Text-to-speech and scene narration
- Directional sound cues
- Haptic feedback integration

### Use Cases
- Independent navigation
- Object identification
- Reading and spatial orientation

---

## 4. Vector Hearing

**Target users:** Deaf and hard-of-hearing users  
**Purpose:** Sound visualization and speech accessibility

### Key Features
- Microphone array around frame perimeter
- Sound direction detection
- Visual sound indicators on lens edges
- Real-time speech-to-text transcription
- Speaker output for voice synthesis (optional)

### Use Cases
- Conversations with hearing individuals
- Environmental sound awareness
- Emergency alerts
- Public and social interaction

---

## 5. Vector Sense

**Target users:** Deafblind users  
**Purpose:** Multimodal perception through non-audio, non-visual feedback

### Key Features
- Advanced haptic signaling
- Vibration-based spatial mapping
- Contextual pattern encoding
- Object and proximity awareness
- AI-driven environmental interpretation

### Use Cases
- Safe navigation
- Human interaction support
- Situational awareness
- Independent movement

---

## 6. Vector Motion

**Target users:** Users with paralysis or severe motor impairments  
**Purpose:** Environmental control and communication

### Key Features
- Eye-tracking interaction
- Head movement detection
- Voice control (if available)
- AI-assisted intent prediction
- Integration with smart wheelchairs and assistive devices

### Use Cases
- Communication
- Mobility assistance
- Smart environment control
- Independence restoration

---
